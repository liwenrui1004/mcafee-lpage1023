# 📊 新旧方案对比

## 架构对比

| 指标 | 旧方案 (Next.js) | 新方案 (原生HTML) | 改进 |
|------|-----------------|-------------------|------|
| **框架依赖** | Next.js 14 + React | 无 | ✅ 零依赖 |
| **文件大小** | ~2-5MB (打包后) | 46KB | ✅ **减少98%** |
