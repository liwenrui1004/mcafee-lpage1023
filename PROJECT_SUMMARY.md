## 📝 使用指南

### 开发环境启动
```bash
# Windows
scripts/start-dev.bat

# Linux/Mac
scripts/start-dev.sh
```

### 内容管理
1. 访问 Strapi 后台: http://localhost:1337/admin
2. 创建管理员账户
3. 配置内容类型
4. 添加内容数据

### 自定义配置
1. 修改 `tailwind.config.js` 调整设计系统
2. 编辑 `types/index.ts` 更新数据类型
3. 调整 `lib/api.ts` 修改 API 逻辑
